工作流

从信息源到你的屏幕，全自动

四个步骤，端到端自动化，无需人工干预。

01

抓取

RSS / Atom GitHub Trending Reddit Hacker News 自定义 API
02

处理

去重、分类，用 AI 大模型生成结构化摘要。

智能去重 内容分类 AI 摘要生成 中英双语 向量嵌入
03

投递

📧 邮件 💬 Slack 🐦 飞书 🔗 Webhook 📄 HTML 页面
04

积累

所有内容归档到本地知识库，向量索引支持语义检索。

SQLite 存储 向量索引 关键词索引 语义检索 偏好学习
